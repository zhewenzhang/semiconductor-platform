# 半导体产业分析平台

专业级半导体产业链数据分析平台，支持公司库、Roadmap、产品分析等功能。

## 功能特性

- 公司库管理 - 完整的半导体公司数据库
- Roadmap 时间轴 - 先进制程/封装技术发展路线
- 产品分析 - 芯片、载板、封装技术对比
- 数据可视化 - ECharts 图表展示

## 技术栈

- React 18 + TypeScript
- Vite
- Ant Design 5.x
- ECharts
- Supabase (后端数据库)

## 快速开始

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建生产版本
npm run build
```

## 部署

支持 Vercel 一键部署
